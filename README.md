# EasyLists
Easy list library is a library for linked lists

## Installation
### Linux and MacOs
```shell
sudo make install
```
### Windows
(I don't know if this actually works and I don't care if it doesn't)
```shell
make
```
## Usage
Please read the example. To compile use something like `g++ file.cpp -lelists`.
